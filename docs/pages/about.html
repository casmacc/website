Containerized Apps for Social Marketing And Customer Communication

Open Source Apps | Best Practices | Fast, Flexible, Cheap

We hope to build a community centered around sharing code and best practice.

This project is for companies who:
  • use social marketing tech
  • need flexible / customizable solutions
  • don't want to build everything from scratch
The benefit for participants is to:
  • reduce risk, time and costs
  • learn from each other
  • build strong professional relationships

Software Tooling

For interoperability, we standardize on common software infrastructure:
Containers Docker Docker, Docker Compose and Docker Application Packages
Messaging NATS Streaming like Kafka, but smaller / faster / simpler
APIs GraphQL simpler and more flexible than restful APIs
