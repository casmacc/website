For companies who:

Participant benefits:
  • reduce risk, time and costs
  • learn from each other
  • build professional relationships

Tooling

For interoperability, we standardize on common software infrastructure:
Containers Docker Docker, Docker Compose and Docker Application
Messaging NATS Streaming like Kafka, but smaller / faster / simpler
Configuration Consul distributed k/v store
Secrets Vault Project for managing secret keys and credentials
APIs GraphQL simpler and more flexible than restful APIs
