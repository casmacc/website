Tooling

For interoperability, we standardize on common software infrastructure:
Containers Docker Docker and Docker Compose
APIs / GraphQL GraphQL GraphQL is a query language for your API
APIs / gRPC gRPC A high performance, open-source universal RPC framework
Messaging NATS like Kafka, but smaller / faster / simpler
Log Routing FluentD log data buffer / router
Configuration Consul distributed k/v store
Secrets Sops for managing secret keys and credentials
K/V Store Redis in-memory data store
RDMBS Postgres relational database
Time-Series DB InfluxDB time-series data
Visualization Grafana visualization, analytics and monitoring
